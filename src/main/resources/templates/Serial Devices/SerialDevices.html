Serial Devices

In this page an administrator of the system can add new Serial Devices that may function as either Interactions, that can work in either stand alone mode or as part of a game (Like Escape The Room), or as Sensors that will be responsible for acquiring readings of relevant environment variables like temperature, humidity, sound, etc...

Add Device

Here an administrator can add a new Serial Device to the system. The serial device can be of either sensor or interaction type.
